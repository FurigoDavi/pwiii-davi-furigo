# React

O **React** é uma **biblioteca JavaScript** criada para construir **interfaces de usuário** (UI), principalmente em **aplicações web**.

## Para que serve

O React é usado para:
- Criar **sites e aplicações web interativas**.
- Construir **interfaces rápidas e dinâmicas**.
- Dividir a interface em **componentes reutilizáveis**.

## Como funciona

O React funciona com **componentes**, que são pequenas partes da interface (como botões, menus ou formulários).  
Esses componentes podem ser reutilizados e atualizados facilmente.

## Exemplos de uso

Algumas aplicações que usam React:
- Redes sociais
- Sistemas web
- Dashboards
- Aplicativos com **React Native** (para celular)

## Resumo

O **React** é uma ferramenta que ajuda programadores a **criar interfaces modernas, rápidas e organizadas para sites e aplicações**.
